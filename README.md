# 💳 Credit Card Fraud Detection

A Machine Learning project that detects fraudulent credit card transactions using classification algorithms. The model analyzes transaction patterns to identify suspicious activities and helps improve financial security by accurately distinguishing fraudulent transactions from legitimate ones.

## 📌 Project Overview

Credit card fraud is a significant challenge in the financial industry. This project uses machine learning techniques to analyze transaction data, handle class imbalance, and build a predictive model capable of detecting fraudulent transactions with high accuracy.

## 🚀 Features

- Data preprocessing and cleaning
- Exploratory Data Analysis (EDA)
- Handling imbalanced datasets
- Feature scaling
- Machine Learning model training
- Fraud detection prediction
- Model performance evaluation
- Confusion Matrix and Classification Report
- Data visualization

## 🛠️ Technologies Used

- Python
- Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

## 📊 Project Workflow

1. Load the credit card transaction dataset
2. Explore and preprocess the data
3. Handle class imbalance
4. Split the dataset into training and testing sets
5. Train the machine learning model
6. Evaluate model performance
7. Predict fraudulent transactions
8. Visualize the results

## 📈 Model Evaluation

The model is evaluated using:

- Accuracy Score
- Precision
- Recall
- F1-Score
- Confusion Matrix
- Classification Report
